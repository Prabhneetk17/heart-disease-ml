1. Data loading, inspection, and cleaning
2. Categorical encoding + StandardScaler normalisation
3. 80/20 stratified train-test split
4. Feature importance analysis (Random Forest)
5. Model training — Logistic Regression, Random Forest, KNN
6. Evaluation — Accuracy, Precision, Recall, F1, Confusion Matrix, ROC-AUC
7. Hyperparameter tuning via GridSearchCV
